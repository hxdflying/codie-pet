# CodiePet Terms

CodiePet is provided as local plugin software under the repository's MIT license.

By using CodiePet, you are responsible for:

- Using only photos and likenesses you have the right to use.
- Reviewing generated avatars before installing workspace behavior.
- Complying with the terms and policies of the Codex product and any image-generation provider used by Codex.
- Removing generated files from the workspace if you no longer want to retain them.

CodiePet does not modify the Codex desktop application UI and does not provide a hosted service.
